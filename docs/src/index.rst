Urban footprinter documentation
=======================

The urban footprinter is a reusable convolution-based approach to detect urban extents from raster datasets, which implements the methods described in the Atlas of Urban Expansion :cite:`angel2012atlas`.

**Citation**: Bosch M. 2020. "Urban footprinter: a convolution-based approach to detect urban extents from raster data". Available from `https://github.com/martibosch/urban-footprinter <https://github.com/martibosch/urban-footprinter>`_. Accessed: DD Month YYYY.
